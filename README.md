# DevOps Web Server Project

## Overview
Automated CI/CD pipeline for a Python web server 
using Docker and Jenkins.

## Tech Stack
- Linux
- Shell Scripting
- Git + GitHub
- Docker
- Jenkins

## Pipeline Flow
1. Code pushed to GitHub
2. Jenkins automatically detects change
3. Old container stopped
4. New Docker image built
5. New container started
6. Changes live instantly!
